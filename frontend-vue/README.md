# E-Commerce Flow Studio

Frontend prototype based on the supplied diagram.

## Stack

- Vue 3
- Pinia
- SQLite in the browser via `sql.js`
- Tailwind CSS
- Heroicons
- Prompt font

```bash

```

## What is included

- Banner Listing -> Checkout -> User flow
- Checkout -> Order -> Warehouse flow
- Full CRUD console for Profile / Order / Product
- Local persistence with browser-side SQLite
